# XLXsFTP客户端 WPF版本

## 概述
- 一个普通的大作业
- 一个普通的小程序
- 一个普通的FTP客户端

## 特性
- 十分简单
- 界面简洁
- 支持 IPv6
- 支持 Unicode
- 支持 Linux / Unix 和 WindowsNT 格式的服务器
- 使用工作线程执行传输队列
